# 🎨 Smart Artisan Assistant

> Empowering artisans with AI-driven production management, analytics, and workflow automation.

---

## 📌 Overview

Smart Artisan Assistant is a modern MERN-stack web application designed to help artisans manage production workflows, track materials, monitor earnings, and improve operational efficiency through AI-powered assistance.

The platform provides dedicated dashboards for:
- 🧑‍🎨 Artisans
- 💰 Accountants
- 🛠️ Admins

This project focuses on solving real-world artisan management problems using scalable full-stack architecture and modern UI/UX principles.

# 🧪 Installation & Setup

## Clone Repository

```bash
git clone https://github.com/your-username/your-repo-name.git
```

---

## Navigate to Project

```bash
cd your-repo-name
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Development Server

```bash
npm run dev
```

---

# 🌐 Environment Variables

Create a `.env` file in the root directory:

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
PORT=5000
```
